COMANDOS UNIX
- cd /c .Para irse a la unidad C o a otra carpeta
- pwd .Muestra ruta actual
- ls .Lista archivos

PARA
git init .Es para inicializar proyecto
git status .Ver estado Archivos
git add -A .Agregar todos los achivos al repositorio. Todos los de la carpeta
git commit -m "pagina en blanco" Se suben los cambios realizados.
git log Lista los cambios que se han subido
git log> logs.txt .Lista los logs que se han hecho
